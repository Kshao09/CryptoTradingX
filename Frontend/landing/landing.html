Frequently Asked Questions

Everything you need to know before you connect your keys.

How do I connect the real WebSocket feed?
Set CONFIG.DEMO=false and provide CONFIG.WS_URL to your server (e.g., wss://api.cryptotradingx.com/quotes). This page auto-switches to live updates.
How do you avoid API quota issues?
We cache hot routes in Redis and batch requests. On client errors (429), we use exponential backoff and user-visible toasts so sessions degrade gracefully.
Stripe & OpenAI setup?
Use environment variables server-side: STRIPE_SECRET_KEY and OPENAI_API_KEY. Client never sees the secrets; this page only pings health status.
